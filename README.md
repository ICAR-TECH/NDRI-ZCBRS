# 🐄 Zebu Cattle Breed Recognition System
## 📌 Overview

This project focuses on developing a scalable and automated system for identifying indigenous Zebu cattle breeds using low-complexity AI models. It targets practical deployment in resource-constrained agricultural environments.

## 🎯 Objective

To accurately classify major milch cattle breeds—Sahiwal, Red Sindhi, Hariana, and Tharparkar—using image-based machine learning techniques with minimal computational requirements.

## 🗂️ Dataset

Images collected from frontal and left-side views

Total 8 classes (breed × view combinations)

Designed to capture real-world variability

## ⚙️ Models Used

Convolutional Neural Network (CNN)

Multi-Layer Perceptron (MLP)

Random Forest (RF)

K-Nearest Neighbour (KNN)

PCA applied for dimensionality reduction

## 📊 Results

CNN: 95% accuracy (best performance)

MLP & RF: ~90% accuracy

KNN: ~88% accuracy

All models showed efficient performance with low computational cost

## 🚀 Key Contributions

Demonstrates effectiveness of lightweight AI models

Enables field-level deployment in smart dairy systems

Supports breed conservation and precision livestock management
